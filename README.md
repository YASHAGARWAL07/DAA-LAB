# DAA Lab Codes

This repository contains my Design and Analysis of Algorithms (DAA) lab programs, assignments, and practice implementations developed during my B.Tech Computer Science coursework.

## Topics Covered

- Brute Force Algorithms
- Divide and Conquer
- Greedy Algorithms
- Dynamic Programming
- Backtracking
- Branch and Bound
- String Matching Algorithms
- Graph Algorithms
- Optimization Problems
- Recursive Algorithms

## Implemented Algorithms

- N-Queens Problem
- Travelling Salesman Problem (TSP)
- Floyd Warshall Algorithm
- Matrix Chain Multiplication
- Fractional Knapsack
- Rabin-Karp Algorithm
- Knuth-Morris-Pratt (KMP)
- Longest Common Subsequence (LCS)
- Brute Force String Matching

## Technologies Used

- C++
- STL
- VS Code
- Ubuntu Linux

## Purpose

The purpose of this repository is to maintain DAA lab work in an organized manner, improve problem-solving skills, and track coding progress using Git and GitHub.
